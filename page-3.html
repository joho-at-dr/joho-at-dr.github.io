const

A function that may look weird is const:

const : a -> b -> a

It takes two values and returns the first (while ignoring the second).

const(1)(2) == 1
const(x)(y) == x

It may seem pointless, but can be useful in combination with other functions.

Filter

The `filter` function may not need much introduction:

filter : (a -> Boolean) -> List a -> List a

It filters a list of values by returning the elements for which the given predicate function returns true. Examples:

filter(isOdd) ([1,2,3,4]) == [1,3]
filter(isEven) ([1,2,3,4]) == [2,4]

What statements should be true?

filter(const(false)) == const([])
filter(const(true)) == id
filter(id) : List a -> List a

Since `id` has type `a -> a` and the predicate function of `filter` must return a Boolean, the input list must be of type `List Boolean`. This will also be true for the output list. So:

filter(id) : List Boolean -> List Boolean
